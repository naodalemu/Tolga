Summary: Conditions and Conditional Statements

Dive into decision-making in Python with an example from a simple game.

  • Scenario: You're writing a game where the player must choose between two doors, leading to different outcomes.
                        
door_choice = input("Choose a door (1 or 2): ")

if door_choice == "1":
    print("Behind door 1 is a friendly dog. You pet the dog and win!")
elif door_choice == "2":
    print("Behind door 2 is a sleeping dragon. You tip-toe past and escape!")
else:
    print("You did not choose a door! Try again.")
Try it yourself >>
  • This example demonstrates the use of if, elif, and else statements to create branching paths based on the player's choice.
  • The program checks for specific conditions (the player's choice) and executes different code blocks accordingly, illustrating the power and flexibility of conditional statements in controlling the flow of a Python program.
